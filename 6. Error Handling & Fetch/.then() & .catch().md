`.then()` and `.catch()` are methods used to handle **Promises** in JavaScript, allowing you to manage **asynchronous operations**.

## ğŸ§  Key Points

| Method     | Description                                                                                        |
| ---------- | -------------------------------------------------------------------------------------------------- |
| `.then()`  | Executes when the Promise is **fulfilled** (resolved). Accepts a callback with the resolved value. |
| `.catch()` | Executes when the Promise is **rejected** (error occurs). Accepts a callback with the error.       |
| Chaining   | Multiple `.then()` can be chained to perform sequential async tasks.                               |
